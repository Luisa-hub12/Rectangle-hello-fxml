# Rectangle-hello-fxml

## Rectangle

### Règles du jeu :

```bash
Nous devons créer un rectangle qui puisse apparaître/disparaitre d'une fenêtre.
On doit aussi pour swipe avec un barre "Slider" Pour pouvoir changer la taille
de notre rectangle. On doit aussi pouvoir cliquer dessus pour le faire changer
de couleur.

```
#### Ce que nous avons fait :

- Le rectangle peut s'afficher/ disparaître quand on clique sur le bouton.
- Le rectangle peut changer de couleur quand on clique dessus :
  - il passe de vert à rouge.
- Nous pouvons changer la taille du rectangle a l'aide d'une barre de swipe.


## Pour tout comprendre :

➡ rend toi dans le fichier HelloController ! 


➡ Rend toi dans le fichier HelloApplication !

➡ Pour finir rend toi dans le fichier hello-view.fxml pour avoir un apperçu du code qui a été fait pour créer mon rectangle.
